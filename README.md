# Neuro-Navigator Pro

An Interactive 3D Cerebral Mapping System built with Next.js 14, Three.js, and Tailwind CSS.
Includes advanced features such as:
- 3D Brain loading and interaction
- Lobe Selection with dynamic data overlay (functions, neurotransmitters, and "what if" scenarios)
- "Simulate Thought" particle system
- Fast search functionality to locate and auto-focus on a specific region.
